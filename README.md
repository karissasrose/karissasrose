# Academic
- [Web Crawler](https://github.com/karissasrose/AssignmentWebCrawler) — A crawler that traverses websites and creates a tree of all links in that website. It recursively visits websites that are reachable through one-another, recording all words listed on those websites to then perform statistical analysis on. It is coded to avoid "traps," such as circular links and websites that can generate an infinite amount of sublinks. Coded as a group of 4 using Python.
